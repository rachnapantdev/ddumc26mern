State : It is used to manage State Locally to the Component.
To use this state in functional component we use "useState()" hook of react .

useState returns an Array with exact two values

1. currentState
2. function to update the current state.

Syntax:
	import {useState} from 'react';
	const [state, setState] =  useState(initalValue);



Routing in React JS :

Routing is used to navigate from one component to another component.

Need to install following in our app:
1. npm i react-router
2. npm i react-router-dom

Types of Routing 
1. Static Routing 
2. Nested Routing 
3. Dynamic Routing 
4. Wild card Routing
