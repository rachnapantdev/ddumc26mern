Structure of Table
<table>
	<thead>
		<tr>
			<th> </th>
			....
		</tr>
	</thead>
	<tbody>
		<tr>
			<td> </td>
			....
		</tr>
	</tbody>
	<tfoot>
		<tr>
			<td> </td>
		</tr>
	</tfoot>
</table> 


<table> </table> :  Container tag of table
<thead> </thead> : Header of the table
<tbody> </tbody> : Body of the table
<tfoot> </tfoot> :  Footer of the table
<tr> </tr> : Table Row 
<td> </td> :  Table Data 
<th> </th> : Table Heading
